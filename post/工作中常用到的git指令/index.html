工作中常用到的git指令

Oct 20, 2020

推送分支并建立连接

指令 作用 说明
git branch -vv 查看本地分支是否与远端分支建立了连接
git push origin <分支名>:<分支名> 仅仅将本地分支推送到远端分支 注意:这个操作只是单纯的把本地的分支推送到远端,尚未建立连接
git push --set-upstream origin <分支名> 将本地分支推送到远端分支并建立连接
git checkout -b <分支名> origin/<分支名> 拉取远程分支到本地（本地与远程建立连接） 本地已经由其他分支了
