环海南东线骑行实记
2024年05月12日
骑行
旅游

前言

自上次环完西线之后，我无心工作😣，一心只想着什么时候把东线给环完。终于趁着五一，赶紧完成这个愿望。

首先我对自行车进行了一番改造，把把手换成了羊角把，这样可以变换骑行手势缓解压力（当然不换羊角把也可以，不过没那么舒服），然后把那可有可无的减震坐管也换掉了，顺便连着坐垫也一起换了。并且去年的西线我是一直用书包背着行李的，这样久了之后肩膀会不舒服，为了不用背书包，于是我买了车前包，还有车尾包（🤬）。为了偶尔的夜骑安全，我又买了一个1200nit的车前灯，之前那个400nit的根本不够用。

为了以防万一我还在家里练习了两次补胎（这也为接下来唯一一次爆胎埋下了伏笔😭）

day0：海口 -> 文昌铺前镇 （33km）

今天是准备6点下班后马上就出发的，结果不知道我在干嘛，犹犹豫豫要不要出发，磨蹭到了9点钟。我犹豫是因为已经四个月没有骑车了（一是懒了，二是因为一月份买了个电动车）。想来想去，还是决定出发。结果没骑个五公里就发现尾包蹭轮，还磨破了一点东西。没辙，还好联系到一家车店还在上班，急忙骑过去安了个后货架，这才出发。到铺前镇已经是凌晨00:30了，让人害怕的是我一进铺前镇，迎接我的是一座乌漆麻黑的城市——几个小时前这里停电了。这时我的前灯也突然没电了，给我吓了一跳，还以为世界末日了。还好镇上有一家宾馆有发电机，我像抓住救命稻草一般住进去，一夜无事~。所谓万事开头难（怎么我去年西线开头的时候那么容易呢，呜呜呜），接下来还有更难的。

day1: 铺前镇 -> 木兰湾 -> 龙楼镇 （98km）

今天是整段东线之旅最难的一天。前半程（至少前40km）全程逆风 + 横风 + 大太阳，偶尔还有上坡，骑到我怀疑人生。

期间我还爆了一次胎，原因是之前在家里练习补胎时，把内胎往里塞的时候夹住了，导致骑着骑着把内胎给磨破漏气了，😑我没想到是这么个爆胎法。还好这次也有陌生大叔出来和我一起补胎，非常感谢～。

而且听了群友的建议之后，我选择了走海南新搞的旅游公路，补给点非常少，可能就三四个补给点。（后悔，现在就是非常后悔）

骑到下午的时候天气稍微好了些，而且大都是平路了，一路吹着风放着音乐，实在是惬意。路上还遇到了一位广西的老表，据他说他是从广西桂林一路骑到海南的，实在是佩服。后边我们一路结伴骑到了龙楼。

晚上吃了只烧鸡，感觉还不错

day2：龙楼镇 -> 博鳌镇（85km）

平平无奇的一天，没有什么新鲜的事情发生。不过途径了一片沿海路段时，可能是因为一路都是暴晒，看到海的那一刻，莫名让人感动。虽然我是海南人，看海也有无数次了，可这次总感觉不一样，有种莫名的感动，彷佛在沙漠之中找到了一片绿洲，让我想起了《肖申克救赎》里的那句台词——“Hope is a good thing”，彼时在我眼前的就是象征希望的海。

day3：博鳌镇 -> 万宁东澳镇（70km）

今天是整趟旅程中最惬意的一天。下午的时候，天空多云，微风习习，我骑在乡间的小道上，路的两旁，要么是植物（当然大多是椰子树），要么田野，还有牛牛。音响里放的是Michael Jackson的《Invincible》专辑，也不知道循环到了哪一首。

后来我坐在后安镇的一个不知名的桥上吹风，看着下面那不知名的河流，坐了许久，这一刻彷佛什么事情都没有眼前的这条河重要，我彷佛感觉到了自我，倘若人生留在那一刻该多好：心头无事，耳边只听见音乐和河流的声音。

day4：东澳镇 -> 陵水县（60km）

今天一出门十来公里就碰到了一个超长的坡（后边我才知道是好汉坡…），然后大盘带小飞的时候把尾拨给卷进轮子里边了，我真是卧槽了，之前都没遇到过这种情况，由于事先也没补充一些修车的知识，我就擅自把尾拨掰了回来，没想到这已经给它照成了不可逆转的损伤。骑了一段路，一直异响不断。然后我又掰了掰，结果把尾钩也掰断了🤣

没办法，只能呼叫合作伙伴货拉拉了。由于我所在位置太偏僻，定位都不知道定位在哪里，后边还是一个好心的大叔用三轮车给我拉到一个好定位的地方，非常感谢！

第一次叫货拉拉，等了大概半个小时终于到了。

day5：陵水县 -> 三亚 （60km）

终于，最后一段旅程了。昨天一直没睡到，今天十点多才出发，吃了个咸豆腐脑+油条（上一次吃还是大学的时候）就出发了。

之后一路无事，反而是刚开始出发的那个咸豆腐脑，让我一路想了很多过去的事情

成功抵达三亚517驿站

后话

损失

这次东线之旅我损失了三样宝贝，呜呜呜，分别是mp3（山灵m0），手机（pixel6pro，内外屏都摔坏了），还有一副大学时候配的眼镜。

mp3是因为车前包没关好，骑着骑着飞出去摔坏了。手机上因为放兜里没放好掉地上裂屏了，眼镜是因为我无聊的时候掰了掰，没想道掰断了。眼镜倒没什么，戴了三四年了本来就想换了，可是mp3和手机，都是因为我的不小心而造成的，如果我能小心一点就好了😣

伤痛

报应还是来了，四个月没骑车，由于对于新的坐垫也没有一个适应过程，东线之旅结束后，我的左手隐隐发疼，没有了握力（连响指都打不了了），给我吓的。后面通过针灸+休息一个星期后才好了些，虚惊一场。

不好好准备就出发，说实话我是在拿我的生命去开玩笑，但有时候想不了那么多了。

结语

去年西线

今年东线

至此，我的海南环岛骑行也就结束了🎉🎉🎉。从2023年5月诞生环岛的想法开始，历经一年终于完成。我想我应该趁着年轻，做一些想做的事情，就酱～
